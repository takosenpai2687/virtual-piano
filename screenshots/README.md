# Screenshots

This folder contains screenshots for the project README.

## How to Add Screenshots

1. Take a screenshot of the virtual piano application
2. Save it as `demo.png` (or any descriptive name like `interface.png`, `playing.png`, etc.)
3. Place the file in this folder
4. Update the main README.md to reference it:

```markdown
![Virtual Piano Screenshot](screenshots/demo.png)
```

## Recommended Screenshots

- **Main interface**: Full view showing the Dynamic Island controls, piano keys, and falling notes
- **Volume control**: Screenshot showing the volume slider popup
- **Playing mode**: Screenshot with notes falling and keys highlighted
- **Mobile view**: Screenshot on mobile device (optional)

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: 1920x1080 or higher
- **Size**: Keep under 1MB for fast loading
- **Optimize**: Use tools like TinyPNG or ImageOptim to compress images
